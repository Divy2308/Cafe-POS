Set up your thermal printer

  1. Download QZ Tray (free, Windows / Mac).
  2. Install and run it — it stays in the system tray.
  3. On this page, choose QZ Tray and click Detect printers.
  4. Select your Epson / TVS / generic 58mm or 80mm USB printer.
  5. Click Test print — you should get a sample slip.

No QZ? Use Browser print and use your OS print dialog to the thermal printer.
